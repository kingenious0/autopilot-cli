# Installation

Install Autopilot globally using npm:

```bash
npm install -g @traisetech/autopilot
```

## Prerequisites

- **Node.js**: Version 18.0.0 or higher.
- **Git**: Installed and available in your PATH.
- **System**: Windows, macOS, or Linux.
